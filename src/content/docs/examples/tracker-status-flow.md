## Example flow

```text
User: I applied to Intermediate Full Stack Engineer at Runn
Assistant: Updated to applied.
User: I'm interviewing for the Intermediate Full Stack Engineer at Runn next Monday morning
Assistant: Updated to interviewing.
User: Any advice to get ready for my interview?
Assistant: Provides targeted prep guidance based on the role context.
```

## What this shows

Tracker state is not just a storage feature. It gives later requests useful continuity.
