## Best use cases

Telegram is best when the user wants a lightweight messaging experience without opening the full web workspace.

## Example requests

```text
How many job applications do I have?
What do you think about this role?
Write a cover letter and save it in my tracker
Save the Intermediate Full Stack Engineer position at Runn in my tracker
```

## What to notice

The transport is different, but the underlying platform and skill boundaries remain the same.
