## What a briefing does

A briefing turns the user profile and current request into a ranked set of job matches.

## Typical steps

1. fetch candidate jobs
2. score them against the profile
3. return the top matches
4. allow follow-up actions such as analysis, cover letters, or tracker updates

## Common follow-up prompts

- `Show me the first 10 matches`
- `Analyze this role`
- `Write a cover letter`
- `Save this to my tracker`
