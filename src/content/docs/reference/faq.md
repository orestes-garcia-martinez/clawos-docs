## Is ClawOS the same thing as CareerClaw?

No. ClawOS is the platform. CareerClaw is the first live skill inside that platform.

## What channels are live now?

Web and Telegram are live in the current MVP direction. WhatsApp is later.

## Does the UI grant Pro access by itself?

No. The platform grants access only after the billing state is verified and the entitlement cache is updated.

## Does ClawOS keep all work in one giant chat thread?

No. The platform is designed around focused skill workspaces and controlled shared context.

## Why is the docs site dark-only right now?

Because the design system is dark-first and the v1 docs goal is stability with minimal maintenance overhead.
