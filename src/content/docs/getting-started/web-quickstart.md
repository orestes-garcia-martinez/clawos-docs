## Web flow

1. Sign in to `app.clawoshq.com`.
2. Enter the CareerClaw workspace.
3. Add preferences and resume text.
4. Start a chat request such as “Run a fresh job briefing”.

## Notes

- Resume handling should stay text-only after extraction.
- Web chat is the richest MVP surface.
- Streaming progress belongs in the web channel.
