## What you get at MVP

ClawOS currently launches with CareerClaw as the first live skill and supports Web and Telegram
as the MVP channels.

## Start path

1. Create an account in the web app.
2. Open the CareerClaw workspace.
3. Add your profile and resume text.
4. Run a fresh job briefing.
5. Upgrade to Pro only when you need premium intelligence features.
