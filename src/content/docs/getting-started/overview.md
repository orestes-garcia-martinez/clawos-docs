## What is live today

ClawOS currently launches with **CareerClaw** as the first live skill and supports **Web** and **Telegram** as the active channels.

## Quick start path

1. Create an account in the web app.
2. Open the CareerClaw workspace.
3. Add your profile and resume text.
4. Run a fresh job briefing.
5. Save or update tracked jobs as your search progresses.
6. Upgrade only when you need the premium intelligence layer.

## Choose your path

### Web first

Use the web app if you want the most complete MVP experience:

- profile editing
- resume text visibility
- guided settings
- richer workspace navigation

Continue with [Web Quickstart](/getting-started/web-quickstart/).

### Telegram first

Use Telegram if you want lightweight access from a messaging channel you already use.

Continue with [Telegram Quickstart](/getting-started/telegram-quickstart/).

## What stays platform-owned

Even when you are working in CareerClaw, the following remain platform concerns:

- user identity
- billing and entitlements
- channel linking
- session continuity
- trust and security boundaries
